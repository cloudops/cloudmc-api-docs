<code>GET /services/<a href="#administration-service-connections">:service_code</a>/<a href="#administration-environments">:environment_name</a>/objects?bucket_name=:bucket_name</code>

Retrieve a list of all objects in a given [environment](#administration-environments) in the specified container.

Mandatory | &nbsp;
------- | -----------
`bucket_name` <br/>*string* | The name of the bucket from which the object must be fetched.

Optional | &nbsp;
------- | -----------
`path` <br/>*string* | Limits the object list to the provided path. (Simulating a bit a folder structure). It does not list "sub-folders".
`show_all` <br/>*boolean* | Combined with the path, it will return also the "sub-folders".
`q` <br/>*string* | Filters the object with the name contains the value of the parameters.

Attributes | &nbsp;
------- | -----------
`id` <br/>*string* | The id of the object .
`name` <br/>*string* | The name of the object.
`displayName`<br/>*string* | The name rendered for display.
`path`<br/>*string* | The path of the object.
`size`<br/>*int* | The size in bytes of the object. It is 0 for object that are used as folders (Generated by UI).
`isNode`<br/>*boolean* | The access type of the container. It can either be public or private.
`bucketName`<br/>*string* | The name of the bucket that the object is part of.
`displaySize`<br/>*object* | Object containing the information to display the size in the UI.
`contentType`<br/>*string* | The mime type corresponding to the content of the file.
`lastModified`<br/>*string* | The last date that the file was modified.

<!-------------------- RENAME OBJECT -------------------->

#### Rename an object (to Complete)

Rename the object to a new name.

```shell
curl -X POST \
   -H "Content-Type: application/json" \
   -H "MC-Api-Key: your_api_key" \
   -d "request_body" \
   "https://cloudmc_endpoint/v1/services/swift/example/objects/exampleFolder/sampleFile.txt?bucket_name=private&operation=rename"

# Request example:
```

```json
{
  "name": "newObjectName"
}
```

<code>POST /services/<a href="#administration-service-connections">:service_code</a>/<a href="#administration-environments">:environment_name</a>/objects/:path_without_container?bucket_name=:bucket_name&operation=rename</code>

<aside class="notice">
Renaming does not exist in SWIFT. It instead consists of the creation of a object and the deletion of the old object.
</aside>

Mandatory | &nbsp;
------- | -----------
`bucket_name` <br/>*string* | The name of the bucket from which the object will be removed. 

Required | &nbsp;
------ | -----------
`name`<br/>*string* | The new name of the container.
